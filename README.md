# Evelyn Nguyen Portfolio

## About This Project

This is my personal portfolio website for CS 463/563 Intro to Web Development.

I made this site to introduce myself, share my experience, show some projects I have worked on, and include a way for people to contact me. It is written in HTML, CSS, and JavaScript

## What’s Included

- Navigation bar
- Header section
- About section
- Experience section
- Projects section
- Contact form
- Scroll and reveal, paw trait effects
- Responsive layout for different screen sizes

## Tools Used

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Bootstrap Icons
- Google Fonts

## How to View the Site

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in a browser.
